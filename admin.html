Como publicar a atualização
Após gerar o dados.json, siga os passos para atualizar o dashboard público
Passo 1
Importe a planilha
Use o modelo oficial, preencha e clique em "Gerar e Baixar dados.json".
Passo 2
Abra seu repositório
Acesse github.com e entre no repositório onde o dashboard está hospedado.
Passo 3
Substitua o dados.json
Clique em Add file → Upload files, envie o novo dados.json e confirme com Commit changes.
Passo 4
Pronto!
Em até 1 minuto o dashboard público reflete os novos dados. Colaboradores não precisam fazer nada.
🔒 Segurança: Colaboradores acessam apenas o index.html que lê o dados.json — somente leitura. Não há upload, formulário ou login na tela deles.
